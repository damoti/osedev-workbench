# Development

```
mkvirtualenv -a `pwd` -p python2 --system-site-packages osedevwb
```

`--system-site-packages` lets us use the same PySide as the freecad binary.

Add `/usr/lib/freecad-daily/lib` to your interpreter paths in PyCharm.
